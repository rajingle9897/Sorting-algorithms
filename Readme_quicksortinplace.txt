Objective: To sort the input elements using Quick Sort.

Input:
User will be asked to enter the number of elements to be sorted. The maximum you can enter is 2,147,483,647. The input will be stored in variable 'n' and an array of elements is created of size 'n' using rand() function.

Output:
Elements will be sorted and execution time will be printed as output.
